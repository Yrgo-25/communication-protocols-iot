# Kommunikationsprotokoll & IoT
Repo för kursen **Kommunikationsprotokoll & IoT** med klassen Ee25, vt26.

---

## Om kursen
Större delen av kursen består av projektarbete med IoT-drivers. Genomgångarna fokuserar på design och implementation av kommunikationsprotokoll.

---

## Struktur

```text
exam/        Tentameninformation och övningsdugga
info/        Kursinformation, schema och examination
lectures/    Föreläsningar, litteratur och övningsuppgifter
project/     Projektbeskrivning, krav och inlämningsinstruktioner
```
